LaTeX2InD

LaTeX2InD is an open-source script for Adobe InDesign that provides a quick way to embed compiled LaTeX code as a PDF within an InDesign file.

It originated as a quick-fix solution to the lack of tools for typesetting mathematics in InDesign, the program that we used during my time at UAM to craft the student magazine.

You can download the script below. In case of doubts or inquiries do not hesitate to contact me.
