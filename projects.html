LaTeX2InD

LaTeX2InD is an open-source script for the Windows version of Adobe InDesign that provides a quick way to include PDF files compiled by either pdflatex, xelatex or lualatex from a LaTeX code input by the user inside an InDesign window.

This script was made to answer for the lack of resources regarding mathematics typesetting in Adobe InDesign. This was made evident to the QED team when the project started, with solutions being very few and with poor results when the complexity of the mathematics involved reached a certain level.

You can download the script below. The installation, usage and commercial use information can be found in the included documentation. In case of doubts, inquiries, or if you are interested in extending the functionalities to Mac/Linux or in generally improving the script, do not hesitate to contact me!
